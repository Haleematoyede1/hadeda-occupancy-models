# Hadeda Occupancy Models

This repository contains the code used to analyse the range expansion of the Hadeda Ibis
(Bostrychia hagedash) in South Africa using single-season, dynamic occupancy models and Sp occupancy models.

## Data availability
The processed datasets used in this study are archived on ZivaHub
(University of Cape Town institutional repository).
